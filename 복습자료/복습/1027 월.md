## Series 안의 원소 접근 - Indexing과 Slicing

#### 내부적으로 관리되는 index(순번)
- 자동으로 배정되는 순번
- 리스트나 튜플의 index와 동일하다

</br>

#### index name (index 이름)
- 명시적으로 각 원소에 지정하는 이름
- 딕셔너리의 key의 역할을 한다
- Series의 index name은 중복될 수 있다

</br>

#### Indexing
- index(순번)으로 조회 : Series.iloc[순번]
- index(이름)으로 조회 : Series.loc[이름]
- index(이름)으로 조회 : Series.[index 이름]
- 팬시 인덱싱 : Series[["a", "b", "c"]]

</br>

#### Slicing
- Series[start index : stop index : step]
- start index : 생략 시 0번 부터 시작
- stop index : 순번일 경우 포함x, 이름일 경우 포함
- step index : 생략 시 1씩 증가 

</br>
</br>

>슬라이싱을 하여 변수에 담고 원소를 변경을 한다면 원본도 같이 변경이 된다. 만약 원소를 복사본만하고 변경하고 싶다면 .copy를 이용하여 Series를 복사해서 만든 뒤 변경하면 된다.

## 원소 단위 연산
- Pandas의 Series나 DataFrame은 연산을 하면 원소 단위 연산
- Series/DataFrame 값을 연산하면 각 원소들과 연산
- Series/DataFrame 끼리 연산하면 같은 위치의 원소끼리 연산

</br>

<code>
<pre>

s1 = pd.Series([80, 90, 100, 50], index=index_name)

## 주요 메소드 및 속성
- head([정수])  :   원소를 정수 개수만큼 조회 (기본값 5)
- tail([정수])  :   원소를 정수 개수만큼 조회 (기본값 5)
- index         :   index 명 조회
- shape, size   :   원소 개수 조회 (메소드가 아니라 변수)
- dtype         :   데이터 타입 조회
- astype(타입)  :   데이터 타입 변경
- isin([리스트]):   Series의 원소가 값리스트 안의 값들 중 같으면 True, 다르면 False인 boolean Series로 반환
- sort_value()  :   값/명을 기준으로 정렬
- sort_index()  :   값/명을 기준으로 정렬
- valuse_count():   고유한 값의 빈도수를 조회
- nunique()     :   고유 값의 개수를 조회
</br>
</br>
- count()       :   결측치를 제외한 원소의 개수
- min()         :   최소값 조회
- max()         :   최대값 조회
- sum()         :   합계
- mean()        :   평균 (결측치 값 제외)
- median()      :   중앙값 (결측치 값 제외)
- std()         :   표준편차 (결측치 값 제외)
- var()         :   분산
- mode()        :   최빈값
- idmax()       :   최대값의 index
- idmin()       :   최소값의 index

</br>

#### mean, std 같은 경우 이상치가 있을 시 값이 이상해질 수 있다. 그래서 median으로 조회하는 방법도 있다.

## 통계에서 변수의 유형

### 범주형 변수
- 의미 : 일정한 범주를 구분하기 위한 이름을 가지는 변수
- 특징 : 가능한 값이 사전에 정해져 있음

#### 명목형 변수 / 비서열 변수
- 의미 : 범주 간 서열 관계가 없는 변수
- 예시 : 성별, 혈액형, 지역, 제품 종류 등

#### 순서형 변수 / 서열 변수
- 의미 : 범주 간 순서 관계는 있으나, 간격이 일정하지 않음
- 예시 : 학점, 직급, 만족도 등

### 수치형 변수
- 의미 : 수량을 표현하는 변수
- 특징 : 수학적 연산이 가능하다

#### 이산형 변수
- 의미 : 정수 단위로만 표현되며, 값들 사이 연속적 구간이 없음
- 예시 : 방문 고객 수, 판매 개수, 결함 건수 등

#### 연속형 변수
- 의미 : 실수 형태로 표현되며, 두 값 사이 무한히 많은 값 존재가능
- 예시 : 키, 몸무게, 온도, 시간, 수익률 등
